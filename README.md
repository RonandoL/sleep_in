# _Sleep In_

##### _Takes in a date and returns the day of the week._

#### By _Ronando Long_

## Description

_Takes in a date and returns the day of the week. If the day is Saturday or Sunday, it tells you you can sleep in._

## Setup

* _plug and play_

## Technologies Used

_Ruby & rspec_
